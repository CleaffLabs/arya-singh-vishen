# Day 4 - How n8n Handles Data

## Overview

Today's session focused on understanding how data is stored and transferred within n8n workflows.

## Topics Covered

* JSON Data Format
* Input and Output Data
* Data Flow Between Nodes
* Item-Based Processing
* Expressions

## Key Concepts

### JSON Format

n8n uses JSON to store and exchange data between nodes.

### Input and Output

* Each node receives input data.
* The node processes the data.
* Output is passed to the next node.

### Data Flow

Data moves through connected nodes in a workflow.

### Item-Based Processing

n8n processes data as individual items, allowing each record to be handled separately.

### Expressions

Expressions allow data from previous nodes to be accessed and reused.

## Key Learnings

* n8n handles data using JSON.
* Data flows from one node to another through inputs and outputs.
* Workflows process data item by item.
* Expressions help access data from previous workflow steps.
